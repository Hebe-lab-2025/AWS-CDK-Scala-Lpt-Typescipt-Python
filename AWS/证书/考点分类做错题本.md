## ✅ Question

**Which cloud service model provides developers with applications, runtime, and development tools?**

### ✔ Correct Answer: C. Platform as a Service (PaaS)

---

## 🧾 English Explanation (first)

**Why PaaS is correct**

PaaS provides:

* application runtime environment
* programming language support
* development frameworks
* middleware
* databases
* deployment tools

Developers only focus on:
👉 writing code
👉 deploying applications

They **do not** manage:
❌ servers
❌ networking
❌ operating systems

**Examples of PaaS**

* AWS Elastic Beanstalk
* Google App Engine
* Heroku
* Azure App Service

---

### ❌ Why the others are incorrect (English)

**A. FaaS — Function as a Service**

* only runs **individual functions**
* fully event-driven
* example: AWS Lambda
  👉 not whole development platform

**B. SaaS — Software as a Service**

* complete **ready-to-use software**
* no development tools
* example: Gmail, Salesforce
  👉 you don’t write code

**D. IaaS — Infrastructure as a Service**

* gives VMs, storage, network
* user manages OS and runtime
  👉 lowest layer, most control, most management

---

## 🇨🇳 中文解释

### ✔ 为什么 PaaS 是正确答案

PaaS 提供：

* 应用运行环境
* 编程语言支持
* 开发框架
* 中间件
* 数据库
* 部署工具

开发者只需要：
👉 写代码
👉 部署应用

不需要管理：
❌ 服务器
❌ 网络
❌ 操作系统

**常见例子**

* Elastic Beanstalk
* App Engine
* Heroku

---

### ❌ 其他选项为什么不对（中文）

**A. FaaS**

* 只运行函数
* 事件驱动
* 典型例子：Lambda
  👉 不是完整开发平台

**B. SaaS**

* 现成软件直接用
* 不提供开发环境
  👉 比如 Gmail、Zoom

**D. IaaS**

* 提供虚拟机、磁盘、网络
* OS 和运行环境都要自己配
  👉 管理工作最多
