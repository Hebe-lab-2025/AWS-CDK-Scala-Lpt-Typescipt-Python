下面是 **MySQL 8** 的示例，往表里 **插入 2 条数据** 👇

---

### 如果你用的是 `users` 表

```sql
INSERT INTO users (username)
VALUES ('alice'),
       ('bob');
```

---

### 如果你用的是 `orders` 表

```sql
INSERT INTO orders (customer_name, status)
VALUES ('Alice', 'NEW'),
       ('Bob', 'PAID');
```

---

### 验证是否插入成功

```sql
SELECT * FROM users;
-- 或
SELECT * FROM orders;
```
