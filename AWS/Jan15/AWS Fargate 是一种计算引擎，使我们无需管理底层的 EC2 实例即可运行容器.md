**翻译：**

> **注意：AWS Fargate 是一种计算引擎，使我们无需管理底层的 EC2 实例即可运行容器。
> 它免除了基础设施管理工作，并为运行容器提供了类似无服务器（serverless）的体验。**


Note: AWS Fargate is a compute engine that enables us to run containers without managing the underlying EC2 instances. 
It does away with infrastructure management and provides a serverless experience for running our containers.
