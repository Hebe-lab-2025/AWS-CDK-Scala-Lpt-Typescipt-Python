**“Launch an Instance” 的准确翻译是：**

> **启动一个实例**
> 或更口语一点：
> **创建并启动一台实例（云服务器）**

---

### 为什么不用“创建”或“启动”其中一个？

因为在 AWS 语境里：

* **Launch** =
  👉 **创建（Create） + 启动（Start）** 一次性完成

也就是说：

* 从 **AMI（模板）**
* 创建一个 **EC2 instance**
* 并立刻让它 **running**

---

### 对应关系（理解很重要）

| 英文                  | 中文含义        |
| ------------------- | ----------- |
| Create instance     | 创建实例（不一定启动） |
| Start instance      | 启动已有实例      |
| **Launch instance** | **创建并启动实例** |

---

### 面试 / 文档级翻译推荐

* 官方 / 技术文档：
  **启动实例**
* 教学 / 解释：
  **创建并启动一台 EC2 实例（云服务器）**

---

### 一句话理解

> **Launch an EC2 instance = 用 AMI 新建一台正在运行的云服务器**
