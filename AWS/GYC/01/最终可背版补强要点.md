## 🔧 二、关键补强 / 校准（只补 5 个高频丢分点）

### 🔐 安全 / 身份（补强）

| 暗号词                       | 条件反射答案         | 备注（考试口径）                                      |
| ------------------------- | -------------- | --------------------------------------------- |
| **DDoS protection**       | Shield         | **大规模 / L7** 时 → CloudFront + Shield Advanced |
| **temporary credentials** | IAM Role / STS | Role 背后一定是 STS                                |
| **block malicious IP**    | NACL / WAF     | IP 级 → NACL；HTTP 攻击 → WAF                     |

📌 **考试习惯**：

* 题干只写 *DDoS* → **Shield**
* 写 *global / edge / L7* → **CloudFront + Shield**

---

### 🌐 网络 / VPC（补强）

| 暗号词                              | 条件反射答案            | 补充                   |
| -------------------------------- | ----------------- | -------------------- |
| **private subnet outbound only** | NAT Gateway       | 放在 **public subnet** |
| **cannot access internet**       | Route Table / IGW | 80% 是 route table    |
| **access S3 privately**          | VPC Endpoint      | 而不是 NAT              |

📌 **一句话防错**：

> NAT 解决“上网”，Endpoint 解决“私网访问 AWS 服务”。

### 🗄️ 数据库 / 缓存（补强）
