RDS 指的是 **关系型数据库服务**，它的全称是：

> **Amazon Relational Database Service**

解释一下关键词：

* **Relational**：关系型（基于表、行、列、主键、外键）
* **Database**：数据库
* **Service**：托管云服务

👉 所以：

* ✔️ RDS 是关系型数据库托管服务
* ✔️ 支持 MySQL / PostgreSQL / MariaDB / Oracle / SQL Server / Aurora
* ✔️ AWS 负责备份、补丁、故障转移等运维
* ✔️ 你主要负责建库、写 SQL、设计表结构

你刚提到的：

* CPU
* 内存
* 存储大小

👉 都是 **RDS 实例规格（Instance Class / Storage）** 的维度。
