## ⚡ 现在就来：30 秒直觉训练（现场示例）

### ⏱️ 模拟真题（你只有 30 秒）

> A web application experiences unpredictable traffic spikes.
> The application must remain responsive and should not be blocked by long-running background tasks such as image processing.
> Which architecture should be used?

---

### 🧠 **直觉定位（你脑子里应该这样走）**

**第 1 秒：抓关键词**

* unpredictable traffic ✅
* long-running tasks ✅
* not blocked ✅

**第 5 秒：自动映射**

* 不阻塞 → **异步**
* 异步 → **解耦**
* 解耦 → **Queue**

**第 10 秒：服务锁定**

* Queue → **SQS**
* Web → **ALB**
* Worker → **EC2 / Lambda**

---

### ✅ **最终答案（不犹豫）**

> Use an **Application Load Balancer** with an **Amazon SQS queue** to decouple the web tier from a worker tier that processes background tasks asynchronously.
