# 2️⃣ 只练“反直觉陷阱”的终极错题本（模板 + 秒杀规则）

## ✅ 你只需要维护这 7 类坑（最提分）

把你错的题号塞进去就行：

```md
# SAA-C03 反直觉错题本（终极版）

## 1) Endpoint vs NAT（该省钱却用 NAT）
暗号：private subnet + S3/DDB/AWS service + cost
秒杀：S3/DDB=Gateway EP；AWS API=Interface EP；别 NAT
错题号：

## 2) Multi-AZ vs Read Replica（HA vs 读扩展）
暗号：failover/HA vs read-heavy/offload reads
秒杀：HA=Multi-AZ；读扩展=Read Replica
错题号：

## 3) Cache vs DB（延迟题不是容量题）
暗号：sub-ms/hot keys/very low latency
秒杀：Cache/DAX first，不要直连数据库
错题号：

## 4) ALB vs NLB（L7 vs L4）
暗号：HTTP/path/host/header vs TCP/UDP/static IP
秒杀：L7=ALB；L4+static IP=NLB
错题号：

## 5) SNS vs SQS vs EventBridge（广播/削峰/事件总线）
暗号：fan-out vs buffer spikes vs rule-based event bus
秒杀：fan-out=SNS；削峰=SQS；事件路由=EventBridge
错题号：

## 6) “EC2 自建”诱导（过度设计）
暗号：EC2 cron / proxy / scripts 出现在选项里
秒杀：只要题干提“managed/low ops/quick”，EC2 基本是错
错题号：

## 7) 网络不通先查谁（别一上来换架构）
暗号：timeout but CPU low / same VPC cannot connect
秒杀：先查 SG/NACL/Route/Target health
错题号：
```
