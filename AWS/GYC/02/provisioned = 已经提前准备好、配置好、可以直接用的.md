## ✅ provisioned 是啥意思？

> **provisioned = 已经提前准备好、配置好、可以直接用的**

中文最贴切的意思：

* **已配置好的**
* **已分配的**
* **已预置的**

---

## 🧠 放到 IT / AWS 语境里怎么理解？

### ❌ 不是

* 临时生成的
* 用的时候才创建的
* 自动弹出来的

### ✅ 而是

* **提前创建**
* **提前分配资源**
* **一直在那里等着你用**

---

## 🧪 具体例子（马上懂）

### 🖥️ EC2

```text
An EC2 instance is provisioned
```

意思是：

> EC2 已经创建并启动好了
> CPU / 内存 / 磁盘已经分配给你

---

### ☁️ 对比 Lambda

* Lambda ❌ 没有 provisioned server
* EC2 ✅ 是 provisioned infrastructure

---

## 🏠 生活比喻（最好记）

### Provisioned

👉 **你提前租好了一套房**

* 房子已经在那里
* 不住也要付钱
* 随时能进去

### 非 provisioned（按需）

👉 **酒店临时订**

* 用一次算一次
* 不用不花钱

---

## 📌 常见搭配（看到就秒懂）

| 原句                         | 实际意思        |
| -------------------------- | ----------- |
| provisioned infrastructure | 已经建好的服务器/网络 |
| provisioned capacity       | 提前买好的算力     |
| pre-provisioned            | 提前准备好       |
